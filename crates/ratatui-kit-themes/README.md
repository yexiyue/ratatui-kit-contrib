# ratatui-kit-themes

Official `ratatui-kit` theme adapters for
[`ratatui-themes`](https://crates.io/crates/ratatui-themes).

This crate converts `ratatui_themes::ThemeName`, `Theme`, and `ThemePalette`
into `ratatui_kit::Palette`. It does not provide another theme provider:
use the converted palette with the core `PaletteProvider`.

![ratatui-kit-themes gallery](https://raw.githubusercontent.com/yexiyue/ratatui-kit-contrib/main/crates/ratatui-kit-themes/assets/gallery.gif)

```rust,no_run
use ratatui_kit::prelude::*;
use ratatui_kit_themes::{IntoKitPalette, ThemeName};

let palette = ThemeName::TokyoNight.into_kit_palette();

let _app = element!(PaletteProvider(palette: palette) {
    Text(text: "The whole subtree uses Tokyo Night")
});
```

By default, conversion preserves the upstream theme background:

```rust
use ratatui_kit_themes::{IntoKitPalette, ThemeName};

let palette = ThemeName::CatppuccinMocha.into_kit_palette();
```

If your application should keep using the terminal background, reset only the
background layers:

```rust
use ratatui_kit_themes::{IntoKitPalette, ThemeName, terminal_background};

let palette = terminal_background(ThemeName::CatppuccinMocha.into_kit_palette());
```

## API

- `palette_from_name(ThemeName) -> Palette`
- `palette_from_theme_palette(ThemePalette) -> Palette`
- `IntoKitPalette::into_kit_palette()` for `ThemeName`, `Theme`, and
  `ThemePalette`
- `terminal_background(Palette) -> Palette`

`ratatui-themekit` is intentionally out of scope so the public model stays
aligned with `ratatui-kit`'s `Palette` / `ComponentTheme` pipeline.

## Gallery

The gallery example shows palette swatches, core components, and
`ratatui-kit-markdown` components in one screen:

```bash
cargo run -p ratatui-kit-themes --example gallery
```

Controls:

- `t` / `T`: cycle through `ratatui-themes` themes
- `b`: switch between theme background and terminal background
- `q` / `Q`: quit

The recording is generated by the package-local VHS tape:

```bash
vhs crates/ratatui-kit-themes/tapes/gallery.tape
```

The tape writes `assets/gallery.gif` and temporary verification screenshots under
`target/vhs-themes-gallery-*.png`.
